서비스 소개서 — 학교(교사·학생·학부모)용 안내

개요
----
이 문서는 학교 현장에서 본 서비스를 사용하는 관점에서 작성된 안내서입니다. 교사·학생·학부모가 실제로 어떤 기능을 어떻게 이용할 수 있는지, 하루 일과에서 어떤 도움이 되는지를 중심으로 정리했습니다.

교사(Teacher) 관점 — 주요 활용 시나리오
-----------------------------------
1) 수업 생성 및 초기 설정
   - 새 수업을 빠르게 생성: 학년도, 학기, 과목명, 강사명, 강의실, 간단한 설명 입력.
   - 방과후 수업 선택 시 학반(학반명·차시 수·스케줄) 옵션을 함께 설정하여 학반 단위로 학생 관리 가능.
   - 수업 생성 시 고유 코드가 발급되어 학생 초대/참여 코드로 사용 가능.

2) 학반 및 학생 관리
   - 학반 생성/수정/삭제: 이름·차시·차시별 요일·교시를 설정하고 학생을 배정.
   - 학반별 학생 목록 관리 및 학반별 출결/공지 분리 운영이 쉬움.

3) 수업 운영(일일 업무)
   - 출결 관리: 해당 수업의 출결을 교사 대시보드에서 바로 체크·수정.
   - 공지 작성: 수업별 공지 작성해 학생(및 학부모)에게 전달.
   - 과제 등록 및 제출 관리: 과제 업로드 → 학생 제출 확인 → 채점·피드백 제공.
   - 학생 평가 기록: 학생별 성취도·평가 기록을 수업 단위로 관리.

4) 수업 편집 및 정보 공유
   - 수업 상세 보기에서 바로 정보 수정(편집 모드) 가능. 수업 정보 수정은 생성 폼과 유사한 입력 방식으로 간편.
   - 수업 상세에서 학반 목록 확인 및 학반 편집(팝업 모달)으로 빠르게 이동.

학생(Student) 관점 — 주요 활용 시나리오
-----------------------------------
1) 수업 참여 및 확인
   - 교사가 제공한 수업 코드를 통해 수업에 참여(또는 학교 포털과 연동된 계정으로 자동 연결).
   - 수업별 공지·자료·과제 목록을 한눈에 확인.

2) 과제 제출 및 상태 확인
   - 과제 업로드(파일/텍스트) 후 제출 상태 확인.
   - 제출물에 대한 교사의 피드백 및 채점 결과 조회 가능.

3) 출결 및 알림
   - 교사가 기록한 출결(결석·지각 등)을 확인 가능(학부모 연동 시 통지).
   - 수업 공지·마감 알림을 통해 과제 기한·행사 안내 등을 실시간으로 확인.

학부모(Parent) 관점 — 주요 활용 시나리오
-----------------------------------
1) 자녀 수업/출결 모니터링
   - 자녀가 소속된 학반의 출결 요약(결석·지각 등)을 확인.
   - 중요한 공지(학사 일정·과제 마감 등)를 수신하여 빠르게 파악.

2) 과제·평가 확인
   - 자녀의 과제 제출 상태 및 교사의 피드백/평가 결과를 열람 가능(권한 제공 시).

3) 소통 채널
   - 교사와의 공지·메시지 기반의 기본 소통 경로 제공(추가 기능으로 메시지 기능 연동 가능).

빠른 시작 가이드 (사용자별)
--------------------------
교사(처음 시작)
1. 로그인 → 대시보드 접속  
2. '수업 생성' 클릭 → 수업 정보 입력 → 저장(수업 코드 발급)  
3. 학반 생성(방과후일 경우) → 학생 배정  
4. 첫 수업 전 공지 작성 및 학습자료 업로드  
5. 출결·과제·평가를 수업별로 관리

학생(처음 시작)
1. 로그인 또는 수업 코드 입력으로 수업 참여  
2. 수업 목록에서 해당 수업 선택 → 공지/과제/자료 확인  
3. 과제 업로드 및 제출 확인

학부모(처음 시작)
1. 학교에서 제공한 접근 권한(또는 계정)으로 로그인  
2. 자녀 소속 학반 선택 → 출결·공지·과제 상태 확인

활용 팁(권장 사용 방법)
---------------------
- 정기 공지 템플릿 활용: 반복되는 안내(수업 준비물, 제출 규칙)는 템플릿으로 관리하면 효율적입니다.  
- 학반 스케줄 사전 등록: 방과후 학반은 학기 시작 전에 차시·요일을 미리 등록하면 출결·알림 관리가 쉬워집니다.  
- 과제 마감 알림 설정: 마감 전 리마인더 공지를 통해 제출율을 높일 수 있습니다.  
- 학부모 공지 권한 설정: 주요 공지(행사·결석 안내)는 학부모에게도 자동 통지되도록 설정하세요.

자주 묻는 질문(FAQ)
-------------------
Q1. 학생이 수업 코드를 분실했어요.  
A1. 교사는 수업 상세에서 수업 코드를 확인 가능하며, 필요 시 수업 정보를 재발급하거나 안내할 수 있습니다.

Q2. 학반에 학생을 추가/제거하려면 어떻게 하나요?  
A2. 학반 편집 모달에서 학생을 선택해 추가·제거할 수 있으며, 학반 삭제 시 관련 출결·기록이 보존되는지 정책을 확인하세요.
